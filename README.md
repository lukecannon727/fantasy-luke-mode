# Fantasy – Luke Mode

Chrome extension for custom Fantasy deck builder

## Install (unpacked from GitHub)

1. **Download the code**
   - Open the repo on GitHub.
   - Click the green **Code** button → **Download ZIP**.
   - Unzip the folder somewhere on your computer (e.g. `Downloads/fantasy-luke-mode-main`).

2. **Load the extension in Chrome**
   - Open Chrome and go to `chrome://extensions/`.
   - Turn **Developer mode** on (toggle in the top-right).
   - Click **Load unpacked**.
   - Choose the **unzipped folder** (the one that contains `manifest.json`, `popup.html`, etc.).
   - The extension should appear in your extensions list and in the toolbar.

3. **Use it**
   - Go to [fantasy.top](https://fantasy.top). Follow the steps in the popup.

## Updating

If you installed from a ZIP, pull the latest code from GitHub, download a new ZIP, unzip it, then in `chrome://extensions/` click the **refresh** icon on Luke Mode so Chrome loads the updated files.
